🎯 ALGORITMA DIAGNOSIS
═══════════════════════════════════════════════════════════════════════════════

Metode: EXPERT SYSTEM BACKWARD CHAINING

Proses:
1. User memilih gejala yang dialami
2. Sistem membaca semua rules dari database
3. Untuk setiap penyakit, hitung persentase kecocokan:
   Persentase = (Gejala Cocok / Total Gejala Penyakit) × 100%
4. Urutkan hasil berdasarkan persentase tertinggi
5. Tampilkan diagnosis utama + diagnosis alternatif

Contoh:
   Input: Sedih, Kurang energi, Gangguan tidur
   
   Depresi Mayor (3/9 gejala) = 33%  ✓ Diagnosis Utama
   PTSD (3/5 gejala) = 60%            ✓ Alternatif
   ...

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

ERROR: "php command not found"
✓ Solusi: Pastikan PHP di PATH Windows atau gunakan full path

ERROR: "Database error / SQLSTATE..."
✓ Solusi: Jalankan: php artisan migrate:fresh --seed

ERROR: "Port 8000 already in use"
✓ Solusi: Gunakan port lain: php artisan serve --port=8001

ERROR: "Blank page / 500 error"
✓ Solusi: 
  1. Generate key: php artisan key:generate
  2. Clear cache: php artisan cache:clear
  3. Check logs: storage/logs/
